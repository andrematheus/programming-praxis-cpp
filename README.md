# README #

This is my attempt at solving [Programming Praxis](https://programmingpraxis.com/) problems using C++.

## Requisites:

C++ compiler (C++14), cmake.

## To Build:

    mkdir build
    cd build
    cmake ..
    make

## To run tests:

- Quietly


    make test
    
- Verbose

   
    make check

## To run binaries for a problem:

    make <problem-name>-run
